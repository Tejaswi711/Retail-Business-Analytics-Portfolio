# Retail Business Analytics Portfolio

## Overview

This project analyzes retail sales data using SQL, Python, and Power BI to identify sales trends, customer behavior, and product performance.

## Tools Used

- MySQL
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI

## Project Workflow

- SQL Data Analysis
- Python Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard

## Dashboard Pages

- Executive Overview
- Product Performance
- Customer Analysis

## Key Insights

- Identified top-performing products and customers.
- Analyzed monthly sales trends.
- Compared revenue by category, region, and customer segment.
- Built interactive dashboards for business insights.
